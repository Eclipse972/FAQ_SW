Ouvrir efficacement un fichier

Un projet peut contenir un nombre &lev& de fichiers. Un moyen de s'en sortir est de filtrer la liste de fichiers.

  1. cliquez dans le menu "Fichier/Ouvrir" ou sur l'icône ouvrir icone à venir
    image de la boite de dialogue à venir
  2. Dans la oite de dialogue en bas à droite sékection le type de fichier: Pièce pour ne s&lectionner que les pièces.
  3. La liste des fichiers diminue dans la zone. S&lectionnez le fichier puis cliquez sur le bouton ouvrir.
    Remarque:
    Pour allez plus vite on peut double-cliquer sur le fichier
  4. La boite de dialogue suivante peut apparaître image à venir. Dans ce cas cliquez sur le bouton "ouvrir en lecture seule".
